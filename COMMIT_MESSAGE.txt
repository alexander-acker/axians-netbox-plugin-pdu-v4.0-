feat!: Migrate Axians NetBox PDU plugin to NetBox v4.0 compatibility

BREAKING CHANGE: This release drops support for NetBox versions prior to v4.0 and Python versions below 3.10.

## Summary
Complete migration of the Axians NetBox PDU plugin from NetBox v2 compatibility to NetBox v4.0+ exclusive support. All plugin functionality and configuration has been preserved while implementing all required v4.0 changes.

## Major Changes

### Python & Packaging
- Update Python support from 3.6-3.8 to 3.10-3.12
- Bump plugin version from 0.0.3 to 4.0.0  
- Set minimum NetBox version requirement to 4.0.0
- Update tasks.py default Python version to 3.11

### Plugin Architecture Migration
- Migrate all imports from `extras.plugins` to `netbox.plugins`
  - Updated: __init__.py, navigation.py, template_content.py
- Remove all NetBox version checking logic (now v4.0+ only)
- Update plugin configuration to target NetBox 4.0+

### Forms & UI Framework Updates
- Remove BootstrapMixin from all form classes per v4.0 requirements
  - Updated: PDUConfigForm, PDUConfigFilterForm
- Remove button color properties from navigation menu items
- Update FilterSets to inherit from NetBoxModelFilterSet

### API Modernization  
- Update API serializers to inherit from NetBoxModelSerializer
- Add brief_fields support to all serializers for v4.0 brief mode
  - PDUConfigSerializer: ["id", "device_type", "power_usage_unit"]  
  - PDUStatusSerializer: ["id", "device", "power_usage"]
- Enable API filters that were previously commented out

### Template System Overhaul
- Remove card-body wrappers around tables (embed directly in cards)
- Update legacy panel classes to card classes
- Remove btn-sm classes from buttons per v4.0 guidelines
- Update icon classes from Glyphicon to Material Design Icons
- Fix HTML syntax error (extra </a> tag) in device_type_pduconfig.html
- Consolidate template files - remove all version-specific variants
- Delete legacy templates: *_3_x.html files

### Code Quality & Cleanup
- Remove unused imports and deprecated functionality
- Simplify view logic by removing version compatibility code
- Clean up template structure and remove legacy files
- Update build system configuration for modern Python versions

## Configuration Preservation ✅
All plugin settings and functionality preserved:
- Plugin name: "axians_netbox_pdu"
